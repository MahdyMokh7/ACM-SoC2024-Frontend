# CA2 — CSS Styling & Animation

## Overview
Building on the HTML foundation, this assignment focused on visual design entirely through CSS — layout, color, hover states, pseudo-elements, and animation — across three distinct mini-projects.

## What's Inside

### Q1 — Animated Gradient Landing Page
A landing page using a four-color `linear-gradient` background, scaled up and animated to slowly pan left-to-right and back via `@keyframes`. Includes uppercase text styled purely through CSS (`text-transform`), hover-triggered button color changes, rounded buttons, and decorative divider lines implemented with `::before` / `::after` pseudo-elements.

### Q2 — Pinterest-Style Image Grid
A scrollable image gallery replicating Pinterest's explore page, including matching hover effects on each image tile — built without a masonry layout, using a straightforward grid arrangement instead.

### Q3 — Personal Portfolio Page
A multi-section single-page portfolio (hero, about, services, call-to-action, footer), styled to a personal aesthetic and theme.

## Key Skills
- CSS `linear-gradient` and background animation with `@keyframes`
- `::before` / `::after` pseudo-elements for decorative content without extra markup
- Hover states and transitions
- Flexbox/Grid layout for image galleries and multi-section pages
- Visual hierarchy and clean layout composition without a framework

## What I Learned
This assignment treated CSS as a design tool rather than just a styling layer — using gradients and animation to add motion and personality, and pseudo-elements to add decorative detail efficiently. Recreating a real product's layout (Pinterest) was a useful exercise in close observation: matching spacing, hover behavior, and visual rhythm rather than guessing at them.
