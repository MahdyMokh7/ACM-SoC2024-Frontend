# ACM Summer of Code 2024 — Frontend Track

Coursework from the **ACM Summer of Code 2024 Frontend Development** track, instructed by **Misagh Mohaghegh** with mentorship from **Sana Sarinavaei**. Across four progressively advanced assignments, this repository documents a path from semantic, SEO-aware HTML, through CSS layout and animation, into the Bootstrap framework, and finally into vanilla JavaScript for DOM manipulation, state persistence, and object-oriented logic.

Each folder is a self-contained, independent project — no shared dependencies, no build step. Open any `index.html` in a browser to see it run.

## Project Summaries

| Project | Focus | Key Skills |
|---|---|---|
| [CA1 — HTML Fundamentals](./CA1-HTML-Fundamentals) | Semantic markup, SEO metadata, accessible forms | Semantic HTML5, meta tags, RTL/i18n, native form validation |
| [CA2 — CSS Styling & Animation](./CA2-CSS-Styling-and-Animation) | Visual design, layout, motion | CSS gradients, `@keyframes` animation, pseudo-elements, Flexbox/Grid |
| [CA3 — Bootstrap Responsive Design](./CA3-Bootstrap-Responsive-Design) | Component-driven responsive UI | Bootstrap 5.3 grid, navbar, cards, floating labels, theming |
| [CA4 — JavaScript DOM & Logic](./CA4-JavaScript-DOM-and-Logic) | Interactivity & application logic | DOM manipulation, event handling, `localStorage`, OOP, sorting |

## Skills Demonstrated

- Semantic, accessible HTML5 and SEO-conscious document metadata
- Responsive layout, both hand-written in CSS and composed with Bootstrap
- CSS animation, custom properties, and light/dark theming
- Vanilla JavaScript: DOM manipulation, event-driven programming, real-time input validation
- Browser storage (`localStorage`) for persisting user preferences across sessions
- Basic object-oriented JavaScript (classes with instance methods)
- Clean separation of HTML, CSS, and JS, and a structured Git/GitHub workflow

## How to View

Every subfolder is a static site with no dependencies. Open the `index.html` file directly in a browser, or serve the folder with a tool like VS Code's Live Server extension for the best experience (especially for the JS-driven projects in CA4).
